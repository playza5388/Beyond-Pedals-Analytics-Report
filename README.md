# Beyond-Pedals-Analytics-Report
## Dashboard
The dashboard can be found in Tableau Public [here](https://public.tableau.com/shared/N3BH9DS3N?:display_count=n&:origin=viz_share_link)
. This dashboard offers a comprehensive overview of profit trends for Beyond Pedals from January 2023 to September 2024. It highlights quarterly profits, overall profit trends, and a detailed monthly profit breakdown by country, product category, and discount percentage.
